Create a fully featured Help Center site (Knowledge Base, Documentation, Wiki, FAQs) with WordPress.

== Description ==

<strong>Create</strong>, <strong>Organize</strong> and <strong>Publish</strong> help content.

A simple tool for creating <strong>Knowledge Base</strong> and <strong>Documentation</strong> sites
for your software products.

<strong>HelpKit</strong> for documentation sites
is what WooCommerce is for eCommerce sites.

<h3>What can I do with HelpKit?</h3>
<ul>
<li>Help Center / Support Sites</li>
<li>Knowledge Base</li>
<li>Documentation</li>
<li>User Manuals</li>
<li>Wiki</li>
<li>FAQs</li>
</ul>

== Installation ==

* In your WordPress admin dashboard navigate to the **Plugins** > **Add New**.
* Search for “WPHelpKit”
* Click the “Install” button.

== Frequently Asked Questions ==

= How to get started? =
Upon activation, the plugin will add a new page named "HelpKit" which will serve as the index for all your help articles.

= Where do I find the Settings / Customization options =
All settings and customization option can be found in Appearance > Customize > Helpkit.

= How to add Help articles? =
In your WordPress administration dashboard, go to HelpKit > Add New

= How to reorder help articles and categories =
Open up the HelpKit page via Appearance > Customize and start draging and dropping help article or categories direcly in the livew preview of the Customizer.
